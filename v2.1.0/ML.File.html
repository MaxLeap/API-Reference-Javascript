Class: File

File

new File(name, data)

A ML.File is a local representation of a file that is saved to the ML cloud.
Parameters:
Name Type Description
name String The file's name.
data Array The data for the file, as either: 1. an Object like { base64: "..." } with a base64-encoded String. 2. an Object url.
