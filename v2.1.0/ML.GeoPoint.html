Class: GeoPoint

GeoPoint

Represents a latitude / longitude point that may be associated with a key in a MLObject or used as a reference point for geo queries. This allows proximity-based queries on the key.

Only one key in a class may contain a GeoPoint.

Example:

  var point = new ML.GeoPoint(30.0, -20.0);
  var object = new ML.Object("PlaceObject");
  object.set("location", point);
  object.save();

Constructor

new GeoPoint()

Creates a new GeoPoint with any of the following forms:
  new GeoPoint(otherGeoPoint)
  new GeoPoint(30, 30)
  new GeoPoint([30, 30])
  new GeoPoint({latitude: 30, longitude: 30})
  new GeoPoint()  // defaults to (0, 0)
